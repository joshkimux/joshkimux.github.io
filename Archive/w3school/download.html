Downloading W3.CSS

Downloading W3.CSS is easy, and you have an intro-level understanding of webdev this will be a pain free operation.


Building a Folder

Let's get started. Before even downloading W3.CSS, we'll need to set up our webapp's home. Go into your computer and create a new folder. Inside that folder go ahead and make two new folders one called img and the other called css. Also, go ahead and download this easy starter template which I've conveniently named "generic" for you. Inside, I've included a basic navigation bar, a footer, and some links to stylesheets.

Download generic here


But wait, isn't this a stylesheet download? Where's the stylesheet?

You, probably

Yes, it is a stylesheet download! But you can also utilize W3 schools by pulling directly from their source website in your code. That's the easy way, but you can also simply pop in the W3.CSS stylesheet directly into your CSS folder and reference it in your html code. i.e. css/W3.css

Download W3.CSS here if you want to append it directly

Getting an IDE

It'll help alot if you have a nice IDE to work on any kind of web-based project. I personally recommend Brackets which is a free to use gift from god. You can download it here. Don't forget about plugins too! Download beautify which auto-organizes your sloppy code here.

Don't like Brackets? Shame on you. But you can also use these other alternatives.

  • Notepad++
  • Dreamweaver
  • Sublime Text
