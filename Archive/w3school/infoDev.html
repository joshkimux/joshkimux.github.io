Writing an Information Page

When a user clicks on a wine card, they'll need to be redirected to an information page! Let's build one now!


Updating the Navigation

As you can see, I've included a navigation page into the file provided in generic.html. We'll need to make a new copy and save it as "info.html." From here, go ahead and reorganize the navigation bar so that it displays the right stuff.



If you're a little confused how this navigation works, give it a look here at W3School.


Setting Up

We're gonna break this part up into segments. Before we begin, remember to start with a w3-container class!

Next step, let's determine what we want to build! For this project, we will provide the following information

  • Picture of the product
  • Quote of what to expect
  • List of ingredients
  • Where to find the product

From this breakdown, we can reach the conclusion that we will need 4 cards to create our new page. Let's build the skeleton first like so.



Inserting the Picture

This is the easy part since we did it once before. Inside your first card, go ahead and insert an img tag with your generic wine's picture. Don't forget to add a styling of width:100% to make it responsive.

Since we also need to label this picture, let's include that in the card as well. To give ourselves a margin, let's make a new w3-container div inside of the card below the picture. Here we'll put our product's name and general info.



Giving a Quote

I personally love using quotes, it's a good way to mix up the visual landscape and gives a focal point for the user to zoom in on. For this product, let's add a quote that will give it some personality and value to the end user.

In order to make a quote in W3Schools you're gonna want to reference this tutorial. Feel free to use any quote type that you want there, but make sure that you change the w3-color code that's provided as the default (usually w3-light-grey). For my example I've used w3-deep-purple to reinforce our existing theme.



Building a List

This is probably the easiest part of the tutorial. All you'll need to do is copy and paste a list format from this tutorial. Once you've done that, simply append/change/delete from the list as necessary.



Adding a Location

This part of the tutorial will only show how to embbed a google maps location into a card. If you want back-end documentation, that's out of the scope of this particular and will depend on where/how you're pulling your user's information from.

To start, you'll need a google maps API key. You can get one here! Once you have your key, simply copy and paste the code google maps provides (the iframe one) into your card. We'll also need to label this, so like before, add in a w3-container with a description attached inside of it.
